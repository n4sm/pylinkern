## Overview

pylinkern is just a gdb plugin and a kernel dev' & exploit environment. For now there is no stable version.

## Installation

```
$ git clone https://github.com/n4sm/pylinkern.git
$ cd pylinkern/core/sbuild/
$ make install
$ make base
```

## Usage

```
$ cd pylinkern/core/sbuild
$ gdb out/vmlinux
